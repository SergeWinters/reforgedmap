About This Project

Project Overview

This interactive map project is designed to assist players of Arma Reforger, a game developed by Bohemia Interactive, by providing tools to plan and manage activities in-game. The map serves as a companion for strategy, navigation, and team coordination.

Credits

  • Game: Arma Reforger by Bohemia Interactive
  • Leaflet.js - Interactive mapping library
  • Leaflet Draw - Drawing tools for Leaflet
  • Leaflet MiniMap - Minimap integration
  • Map resources - Extracted and customized by Roberto for this project

Roadmap

  • Version 1.0 - Core map functionality with markers
  • Version 2.0 - Advanced markers, POI search, and interactive features
  • Version 3.0 - Drawing tools, compass integration, and zoom control
  • Version 4.0 - Minimap integration and multi-map support
  • Future Plans:
    • Mobile application for offline map planning
    • Real-time synchronization with team members
    • Customizable themes for personalization
    • Enhanced feedback and collaboration tools
