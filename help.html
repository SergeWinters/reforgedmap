Features Overview

This interactive map offers several tools and features designed to enhance your planning and navigation experience in Arma Reforger.

Available Features

  • Markers: Place customizable markers on the map, such as enemies, respawn points, and loot.
  • Remove Markers: Click on markers to delete them, or use the "Remove All" button to clear the map.
  • Undo Last: Reverse your last action to remove the most recently placed marker.
  • Drawing Tools: Draw shapes (lines, circles, rectangles) on the map for planning strategies.
  • POI Search: Search for predefined Points of Interest (POI) using the search bar.
  • Minimap: A small overview map is available on the right side, showing your location within the full map. You can interact with it to move the main map.
  • Compass: Rotate the map to align with different directions using the compass control.
  • Zoom Control: Adjust the zoom level and display the current zoom in real time.
  • Map Selector: Switch between different maps seamlessly using the dropdown menu.

Quick Tips

  • To add a marker, select the desired type from the menu and click on the map.
  • Use the minimap to quickly move around the main map by clicking on the area you want to focus on.
  • Drawing tools can be enabled or disabled from the menu for planning complex routes or strategies.
  • Rotate the map using the compass if you need a different perspective.
